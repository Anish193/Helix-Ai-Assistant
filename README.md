# Helix - Personal AI Assistant

A JARVIS-like AI assistant that leverages OpenAI's language models to provide intelligent voice/text interaction with smart home automation, reminders, and general AI capabilities.

## Features

### Core Capabilities
- **Natural Language Processing**: Uses OpenAI's GPT models for intelligent conversation and task understanding
- **Voice Control**: Speak naturally to interact with the system (voice input/output)
- **Smart Home Control**
  - TV Control: Turn on/off, change channels, adjust volume
  - Extensible device support for future automation
- **Reminder System**: Set, manage, and receive reminders
- **General AI Tasks**: Answer questions, provide information, assist with various tasks

### Additional Features
- Real-time task execution
- Context-aware responses
- Persistent reminder storage
- Multi-device compatibility (with proper setup)

## Prerequisites

Before you begin, ensure you have:

- Python 3.8 or higher
- An OpenAI API key
- Smart TV with compatible control interface (HDMI-CEC, network API, or IR control)
- Microphone and speaker (for voice interaction)
